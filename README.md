# Getting Started
```
$git clone https://github.com/silvestreluna/Favorite-Places
```

# Prerequisites
```
  $npm install -g http-server
```
# Running
Browse to Favorite Places directory and run http server.
```
$hs -p 5000
```
Then, in your browser,copy and paste ```localhost:5000```
